§醬燒茄子+烤鮭魚§

§醬燒茄子+烤鮭魚§

🍅食材
茄子兩條/蔥/蒜/醬油適量/糖適量
全聯鮭魚/鹽適量/糖適量
🍳步驟
STEP1⃣
茄子切塊後泡醋水 防止變色 （網路上說的不知道有沒有用啦） 要用時再沖淨瀝乾
STEP2⃣
水煮茄子 煮至稍微變軟（不到軟爛）後撈起 （網路上是說要過油但我覺得那樣太油了）
STEP3⃣
熱鍋加油下蔥蒜炒香 下茄子後加入醬油和糖 加一點水 炒至收汁 茄子呈現稍微軟爛狀態 乘盤 結束！
👆茄子
👇鮭魚
STEP4⃣
鮭魚除刺 均勻抹鹽 朝上那面灑糖 烤箱烤約5-6分鐘
STEP5⃣
取出後擠點檸檬汁 結束~

§Burger🍔§

§Burger🍔§

🍅食材
漢堡麵包 / 牛肉 / 蘿美 / 洋蔥 / 蛋 / 涼拌小黃瓜
醃肉用- 醬油兩匙 米酒一匙 糖一匙 胡椒適量 薑泥
🍳步驟
STEP1⃣
醃肉 至少醃一小時
STEP2⃣
洋蔥切丁 蘿美清洗後用開水沖過濾乾
STEP3⃣
熱鍋加油 下洋蔥 炒出香味後下牛肉炒至收汁 起鍋備用
STEP4⃣
將漢堡麵包放進烤箱烤約3-5分鐘 同時煎顆荷包蛋
STEP5⃣
組裝~ （涼拌小黃瓜是之前做的！）

§佛蒙特甜味咖哩§

§佛蒙特甜味咖哩§

🍅食材
佛蒙特咖哩 甜味
馬鈴薯 / 紅蘿蔔 / 蘋果 / 肉片 / 雪白菇 / 蔥蒜洋蔥
🍳步驟
STEP1⃣
馬鈴薯紅蘿蔔蘋果切塊 洋蔥切條（其實隨便）
STEP2⃣
熱鍋加油下洋蔥蒜頭紅蘿蔔炒香之後加馬鈴薯 加水滾個10-20分鐘
STEP3⃣
下肉片 雪白菇 蘋果 咖哩塊 拌勻之後再滾個10-20分鐘
STEP4⃣
撒蔥花 然後就可以起鍋了~
